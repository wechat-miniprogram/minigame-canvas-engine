# Lobster Reskin Plan (Phase 1)

## Goals
- Keep core mechanics unchanged (movement/collision/score loop)
- Replace visual/audio identity with original lobster theme
- Prepare compliant ad integration hooks (no intrusive changes)

## Reskin Scope
1. Character: replace player sprite with lobster mascot (3 skins)
2. Obstacles: replace bullets/blocks with seaweed/pot/foam objects
3. UI: score -> "虾币", life -> "耐久"
4. Palette: deep sea blue + coral red + gold accents
5. Audio: soft click/jump/hit/clear sounds with ocean ambience

## Non-Functional
- Keep FPS stable on low-end devices
- Asset bundle size growth <= 2MB in Phase 1
- Keep naming and attribution docs for all assets

## Acceptance
- One full game round playable with no crash
- New theme appears in start/battle/result scenes
- Existing core logic and rules unchanged
